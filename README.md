# 🐍Python Basics

A collection of Python practice scripts covering core programming 
fundamentals, written while learning Python from scratch. 

# Topics Covered
- If-Else conditions
- For and While loops
- Functions
- Strings and F-Strings
- Lists, Sets and Dictionaries
- Match Case statements
- File Handling
- Logic Programs
- Graphics Programs
- Type Casting

## Tech Stack
- Python 3

## How to Run
Each file is independent. Just run any file directly:
```bash
python if-else.py
python functions.py
python file-handling.py
```

## Purpose
These scripts were written as part of learning Python fundamentals 
from scratch. Every developer start somewhere! 🚀
